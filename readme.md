# E-Commerce MERN Project

> Front end creating on react,redux and backend node express mogodb

## Usage

Rename "config/config.env.env"
to "config/config.env" and put your values in MONGO_URI

## Install Dependencies

```
npm install
```

## Run App

```
npm run dev
```
